CSS Specificity

Specificity คือ ???

หากมี CSS Rules มากกว่า 1 อันชี้ไปที่ Element เดียวกัน เบราว์เซอร์จะตัดสินใจว่า จะใช้ Rule อันไหน โดยใช้กฎของ Specificity ดูว่าอันไหนมีค่าคะแนนมากที่สุด ก็จะใช้ Rule นั้น

** ถ้า Specificity มีค่าเท่ากัน ให้ Rule ที่ประกาศลำดับต่อมา มีค่าสูงกว่า

** ถ้า มี !important ที่ Rule ไหน ให้ถือว่าอันนั้นมี Specificity สูงสุด (สูงกว่า inline styling)

การคำนวณค่า Specificity

กำหนดตัวเลข 4 หลักคือ (0,0,0,0) โดยหลักแรกจากทางซ้าย มีค่า Specificity สูงสุด

(1) หาก Rule เป็นแบบ inline styling จะได้คะแนน (1,0,0,0) ทันที

หากไม่ใช้ inline styling ให้คำนวณด้วยวิธีการดังนี้

(2) หาก Selector นั้นมี ID ให้บวกคะแนน (0,1,0,0) ในแต่ละ ID

(3) หาก Selector นั้นมี Class, Psuedo-class หรือ Attribute ให้บวกคะแนน (0,0,1,0) ในแต่ละ Class, Psuedo-class หรือ Attribute

(4) หาก Selector นั้นมี Element ให้บวกคะแนน (0,0,0,1) ในแต่ละ Element

ตัวอย่างการคำนวณ Specificity

<p style="color: blue;">Hello</p>
ค่า Specificity คือ (1,0,0,0) ตามกฎข้อ (1) 

section#hdr div.main input
ค่า Specificity คือ (0,1,1,3) เพราะ
มี Element 3 อัน คือ section,div,input (0,0,0,3)
มี class อยู่ 1 อันคือ .main (0,0,1,0)
มี Id อยู่ 1 อันคือ #hdr (0,1,0,0)
รวมกันได้ (0,1,1,3)

ตัวอย่างการ Apply Rules

  h2 { color: red; } // <= Specificity: (0,0,0,1)
  section h2 { color: yellow; } // <= Specificity: (0,0,0,2)

h2 จะเป็นสีเหลือง เพราะมี Specificity สูงกว่า 

  h2 { color: red; }
  h2 { color: blue; }

h2 จะเป็นสีน้ำเงิน เพราะ ถึงแม้จะชี้ไปที่ h2 เหมือนกัน แต่ Specificity ที่ประกาศภายหลังมี Specificity สูงกว่า
